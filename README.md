# Command Line Tools

Learning project for RUST. Implement the basic command line tools for Linux.

 - cat

   Learn to parse the command line and basic file manipulation.
   Learn how to implement unittest and functional tests ussing assert_cmd, assert_fs and predicates.
   Basic Error handling.
 
 - md5sum

   Force code structre to be easier to unnitest. 
   Basic patern matching.
